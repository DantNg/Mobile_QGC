# QGroundControl - Custom Build

> **⚠️ This is a custom fork of [QGroundControl](https://github.com/mavlink/qgroundcontrol), modified to support building and running on Windows 11 and Raspberry Pi.**

This repository is based on the official QGroundControl source code and has been customized with the following changes:

- **Windows 11 build support**: Optimized build configuration and dependencies for Windows 11 with MSVC 2022.
- **Raspberry Pi build support**: Adapted for cross-compilation and native building on Raspberry Pi (ARM).
- **DJI-style UI redesign**: Modern, clean user interface inspired by DJI Fly / DJI Go apps with dark theme, cyan accent colors, and telemetry HUD overlay.

---

### 📋 *About*

*QGroundControl* (QGC) is a highly intuitive and powerful Ground Control Station (GCS) designed for UAVs. Whether you're a first-time pilot or an experienced professional, QGC provides a seamless user experience for flight control and mission planning, making it the go-to solution for any *MAVLink-enabled drone*.

---

### 🎯 *Target Platforms*

| Platform | Status | Notes |
|----------|--------|-------|
| **Windows 11** | ✅ Supported | MSVC 2022, Qt 6, Ninja build system |
| **Raspberry Pi** | ✅ Supported | ARM build, native or cross-compilation |

---

### 🌟 *Key Features*

- 🕹️ *Full Flight Control*: Supports all *MAVLink drones*.
- ⚙️ *Vehicle Setup*: Tailored configuration for *PX4* and *ArduPilot* platforms.
- 🎨 *DJI-Style UI*: Custom modern interface with dark theme and telemetry HUD.
- 🔧 *Open Source*: Customize and extend the software to suit your needs.
